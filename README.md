# Rick and Morty GraphQL Episodes App

A Next.js + Apollo Client application that queries the Rick and Morty GraphQL API and displays episode information with pagination.

## Tech Stack
- Next.js with TypeScript
- Tailwind CSS for styling
- Apollo Client for GraphQL queries

## Features
- Fetch and display Rick and Morty episodes using the official GraphQL API
- Pagination support for navigating through episodes
- Global error handling using a custom `ErrorBoundary` component
- Responsive UI built with Tailwind CSS
